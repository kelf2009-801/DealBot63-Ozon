## 🤖 Команды бота

| Команда | Что делает |
|---------|-----------|
| `/start` | Приветствие и список команд |
| `/parse [ссылка]` | Спарсить товар по ссылке |
| (просто ссылка) | Автоматический парсинг — просто кинь ссылку |
| `/prices` | Показать все отслеживаемые товары |
| `/stats` | Статистика |
| `/export` | Экспорт списка товаров |
| `/untrack [номер]` | Удалить товар из отслеживания |
| `/help` | Справка |
| `/tariff` | Выбор тарифа |
| `/admin` | Админ-панель (с паролем) |

### 🔹 Reply-кнопки (рядом с полем ввода)

```
[📋 Мои товары] [📊 Статистика]
[💰 Тариф]
```

## 🐳 Docker

### Управление ботом

| Команда | Что делает |
|---------|-----------|
| `docker ps` | Показать запущенные контейнеры |
| `docker logs ozon-price-bot` | Логи бота |
| `docker logs ozon-price-bot --tail 50` | Последние 50 строк логов |
| `docker restart ozon-price-bot` | **Перезапустить бота** |
| `docker stop ozon-price-bot` | Остановить |
| `docker start ozon-price-bot` | Запустить |
| `docker exec -it ozon-price-bot bash` | Зайти внутрь контейнера |

### После изменения кода (main_full.py или Dockerfile)

```bash
cd D:\bot_ozon
docker compose build
docker compose up -d --force-recreate
```

### После изменения настроек (config.py)

Достаточно просто перезапустить (без пересборки):

```bash
docker restart ozon-price-bot
```

## ⚙️ Настройка

Все настройки в файле **`D:\bot_ozon\config.py`**:

```python
# Пароль для админки
ADMIN_PASSWORD = "1434"

# Твой Telegram ID (кто может заходить в /admin без пароля)
ADMIN_IDS = [378061707]

# Тарифы
TARIFFS = {
    "free": {
        "name": "Бесплатный",
        "max_products": 3,           # ← сколько товаров можно добавить
        "check_interval": 3 * 60 * 60  # ← интервал проверки (3 часа)
    },
    "premium": {
        "name": "Премиум",
        "max_products": 20,
        "check_interval": 30 * 60    # 30 минут
    },
    "vip": {
        "name": "VIP",
        "max_products": 100,
        "check_interval": 10 * 60    # 10 минут
    }
}
```

**Как менять:**
1. Открыть `config.py` в блокноте
2. Поменять цифры
3. Сохранить
4. `docker restart ozon-price-bot`

### Секреты (токен)

Токен бота лежит в **`.env`** — его не светим, он секретный.

## 🔧 Устранение проблем

### Бот не отвечает

```bash
# Проверить жив ли контейнер
docker ps

# Посмотреть логи
docker logs ozon-price-bot --tail 20

# Если статус "Restarting" — ошибка при старте
docker logs ozon-price-bot
```

### Ошибка "Unauthorized"

Сдох токен в `.env` — обнови через BotFather и пропиши заново.

### Не могу зайти в /admin

Пароль в `config.py` → `ADMIN_PASSWORD`. Если забыл — открой файл и посмотри.

### Бот тупит на парсинге

CloakBrowser грузит страницу 5-30 секунд. Это нормально. Если реально долго — интернет или сам Ozon тормозит.

## 💡 Советы

- После перезагрузки компа бот стартует **сам** (Docker Desktop в автозагрузке)
